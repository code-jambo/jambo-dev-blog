## installation 

```[bash]
git clone https://github.com/code-jamo/jambo-dev-blog
```

```[bash]
cd jambo-dev-blog
```
```[bash]
comoser install
```
```[bash]
cp example.env .env
```
configure db Connection Credentials
```[env]
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=credo_saas
    DB_USERNAME=root
    DB_PASSWORD=
```

Run Migrations

    php artisan migrate

Serve The Blog
    
    php artisan serve
